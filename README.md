# Utensils Shop

Modern kitchen utensils e-commerce demo with a React + Vite frontend and Node/Express backend.

## Quick start

```bash
# backend
cd backend
npm install
npm run dev

# frontend (new terminal)
cd ../frontend
npm install
npm run dev
```

The frontend expects the API at `http://localhost:5000/api`. Override with `VITE_API_URL`.

### Default accounts

- Customer: `ava@example.com` / `cookbetter`
- Admin: `admin@utensils.shop` / `cookbetter`
